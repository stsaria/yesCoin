# YesCoin
このリポジトリにアクセスしていただいて、お誠にありがとうございます！
## YesCoinとは
YesCoinはある団体(YINST)の、仮想通貨です。
ビットコインのように良い作りはしていませんが、
この仮想通貨が広まるとうれしいです！
## 動作要件
- Python3(~3.9ぐらいがおすすめ)
- pip3
- git
## インストール
YesCoinをダウンロードします
```
~$ git clone https://github.com/stsaria/yesCoin
```
ライブラリをインストールします
```
~yesCoin$ pip3 install -r requirements.txt
```
YesCoinを実行します！
```
~yesCoin$ python3 src/main.py
```
